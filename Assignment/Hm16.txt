What is a Package Manager (e.g. NPM, Composer)?
How does it play a role in software development?

A Package Manager is a system/tool that automates the process of process of installing, upgrading, or configuring.
Having a project manager such as NPM can provide others an accessbility to a project 
so that they can access to the project by just running NPM, especially when doing teamwork.
